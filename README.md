# Forex Lot Size Calculator

[![Flutter](https://img.shields.io/badge/Flutter-3.0-blue?logo=flutter&logoColor=white)](https://flutter.dev/)
[![Dart](https://img.shields.io/badge/Dart-3.0-blue?logo=dart&logoColor=white)](https://dart.dev/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A Flutter app to calculate Forex lot sizes including **XAU/USD (Gold)** and **XAG/USD (Silver)**, with a clean dark theme and blurred background.

---

## **Features**

- Calculates:
  - Lot Size  
  - Risk Amount  
  - Position Value
- Supports major Forex pairs and metals (XAU/USD, XAG/USD)
- Dark theme with blurred background image
- Clean and easy-to-use UI
